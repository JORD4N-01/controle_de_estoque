# 🧠 Visão do Projeto

O projeto consiste na criação de uma API simples de controle de estoque utilizando Python com Flask, permitindo:

- Cadastro de produtos
- Registro de entradas de estoque
- Registro de saídas de estoque
- Listagem de dados
