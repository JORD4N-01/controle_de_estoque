# MARI - Modelagem (POO)

## Objetivo
Responsável por criar a base do sistema utilizando Programação Orientada a Objetos (POO).

## Tarefas

- Criar a classe `Livro`
  - Atributos:
    - id
    - titulo
    - autor

- Criar a classe `Emprestimo`
  - Atributos:
    - id
    - livro_id
    - usuario

- Definir o relacionamento:
  - Um empréstimo deve estar ligado a um livro

## Observações

- O código deve ser simples e organizado
- Não precisa usar banco de dados, apenas estrutura de classes
- Manter padrão de nomes definido pelo Tech Lead

## Entregável

- Arquivos:
  - models/livro.py
  - models/emprestimo.py
